Start a new session. Read the TODO system to understand current state.

## Step 1: Read Project Dashboard
1. Read `docs/PROJECT_TODO.md` (index)
2. Find the current active phase from the Phase Dashboard table
3. Check the "Current Focus" section for active task and blockers
4. Check most recent entry in Work Log

## Step 2: Read Active Phase File
1. Read the active phase file from `docs/phases/`
2. Check the Progress Summary table for section-level completion
3. Find tasks marked with 🔄 (in progress)
4. Identify next pending tasks

## Step 3: Check Active Work Documents
1. List folders in `docs/work/`
2. For each folder, read `checklist.md` — find any with status IN PROGRESS
3. If found, include in report as "Active Work Session"

## Step 4: Read LESSONS.md
- Recall recurring mistake patterns (especially guessing and skipping verification)

## Step 5: Report

```
Current State
- Phase: [active phase + progress %]
- In progress: [tasks from phase file]
- Last work: [most recent Work Log entry]

Active Work Session
- [task-name]: [X/Y completed] — last updated [date]
- Read: docs/work/[task-name]/checklist.md to resume

Next Up
- [next pending task from active phase file]

Watch Out
- [relevant patterns from LESSONS.md]
```

## Rules
- Read BOTH the index (PROJECT_TODO.md) AND the active phase file
- Only report AFTER actually reading files (no guessing)
- If files don't exist, say "file not found"
- After reporting, suggest next task and ask "Ready to start?"
- Do NOT start work without user confirmation
